Detailed Description

Interface with an Bosch BMP280 pressure sensor over SPI. For more information refer to the datasheet: https://www.bosch-sensortec.com/media/boschsensortec/downloads/datasheets/bst-bmp280-ds001.pdf

uint32_t BMP280::compensate_pressure ( int32_t  uncomp_pres)
private

Calculate pressure from measurement and compensation parameters.

Parameters
uncomp_presRaw pressure measurement.
Returns
Pressure in Pa as unsigned 32 bit integer in Q24.8 format (24 integer bits and 8 fractional bits). Output value of “24674867” represents 24674867/256 = 96386.2 Pa = 963.862 hPa
